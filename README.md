# WorkMate 🚀

WorkMate is a modern freelancer productivity platform for tracking time, managing projects, generating invoices, and analyzing earnings.

## Features

- Client Management
- Project Management
- Task Tracking
- Live Timer
- Manual Time Entry
- Earnings Calculator
- Multi-Currency Support
- Invoice Generator
- PDF Export
- Reports (Daily, Weekly, Monthly, Yearly)
- Analytics Dashboard
- Dark Mode
- Mobile Responsive UI

## Tech Stack

- React + Vite
- TypeScript
- Tailwind CSS
- Shadcn UI
- Supabase (Auth + Database + Storage)
- Recharts
- React Hook Form + Zod
- pdf-lib
